# calcserv
Веб-сервис, выполняющий функцию решения квадратных уравнений.
## Функциональность
- Решение уравнений вида: ax² + bx + c = 0
- Ввод коэффициентов через виртуальную клавиатуру
- Интерактивный интерфейс с подсветкой активного поля

## Установка
1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Используйте калькулятор:
   - Вводите коэффициенты a, b, c
   - Нажимайте "Следующий" для перехода между полями
   - Нажмите "Решить" после ввода всех коэффицентов для получения ответа

## Технологии
- HTML5, CSS3, JavaScript
